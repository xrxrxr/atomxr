# Atomxr
script qui install atom sur les debian/ubuntu 
avec les sources donc updates de atom 
install mes plugins favoris 



### Installation
```
git clone https://github.com/xrxrxr/atomxr.git
```

```
cd atomxr 
```
```
sh ./script.sh 
```
